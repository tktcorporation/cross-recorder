# プッシュ前のローカルチェック

コミット・プッシュの前に、lint と型チェックをローカルで実行して pass を確認すること。
**チェックが fail している状態でプッシュしてはいけない。**

## 実行するコマンド

```bash
# 1. 型チェック
bun run typecheck

# 2. lint
bun run lint

# 3. テスト（変更に関連するテストがある場合）
bun run test
```

## ルール

- `bun run typecheck` と `bun run lint` は**プッシュ前に必ず実行**する
- 既存の（変更と無関係な）エラーは許容するが、自分の変更で新たなエラーを増やしてはいけない
- fail した場合は修正してから再度チェック → パスを確認 → プッシュ
- 環境依存で実行できない場合（依存パッケージ未インストール等）は、その旨をユーザーに報告してから判断を仰ぐ
